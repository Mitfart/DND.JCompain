Правила:

  1. У каждого оружия (стартовое Простое - 0, Воинское - 1) есть своё количество Очков, которые тратятся по следующим таблицам
  2. Примерная цена оружия расчитывается по формуле, но может определятся мастером в зависимости от Сюжета:
    (10 + 25 x Количество Очков) x Количество Свойств^2
